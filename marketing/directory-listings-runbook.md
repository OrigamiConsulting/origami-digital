## Directory 1 — Clutch.co (10 min)

1. Open https://clutch.co/get-listed
2. Select **Basic** (free) profile
3. Sign up — choose **Sign in with LinkedIn** if possible (the new Origami Digital company page would be ideal here — otherwise use Google SSO with the Origami Consulting Group Gmail)
4. Fill in the Company Profile form using the values above
5. **Project minimum:** R30,000 / $1,500
6. **Hourly rate range:** $25 - $80
7. **Service lines:** tick the full list above
8. **Industry focus:** tick the full list above
9. **Client focus:** enter the 40/50/10 split
10. Paste the **long description** into the About field
11. **Add portfolio (optional but valuable):** add 2-3 portfolio entries
    - Project name + client + 3-4 sentence description + screenshot URL
    - Candidates: Origami Finance, Origami Pay, ImpactRoots, Origami EV Connect
12. **Submit for review** — Clutch reviews new basic listings within 5-10 business days
13. Clutch will email you with verification steps — follow them
