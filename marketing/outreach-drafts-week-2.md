## General response playbook

(Same as Week 1, repeated here for quick reference.)

**They reply asking for the writeup:**
Run a proper audit via the PageSpeed + schema validator, produce a 1-page PDF with the top 5 findings ranked by impact. Attach to the reply. Do NOT pitch a project in the same email — let the document do the work.

**They reply positively, want to talk:**
Offer a 20-minute call (Calendly link to /contact). Come prepared with screenshots, the PageSpeed results, and 3 quick-win recommendations. Only mention price if they ask.

**They reply defensively ("we already have a developer"):**
"Totally understand — wasn't suggesting a replacement. If they're across the [FAQPage schema / Article schema / whatever the specific finding was] issue already, great. If not, feel free to pass this note along. Cheers."

**They ignore:**
ONE follow-up after 10 days. Link the blog post that covers the issue you flagged:
- Schema / GEO → https://origami-digital.co.za/journal/how-to-get-recommended-by-chatgpt
- Redesign / performance → https://origami-digital.co.za/journal/signs-website-needs-redesign-2026
- Website cost questions → https://origami-digital.co.za/journal/website-cost-south-africa

Then drop it.
